What is Mircosoft XNA?

Mircosoft XNA is a freeware set of tools with a managed runtime environment that Mircosoft devolped to facilitate video game development. XNA is based on .NET framework, with version that run on Windows and Xbox 360. XNA Game Studio can help develop XNA games.
